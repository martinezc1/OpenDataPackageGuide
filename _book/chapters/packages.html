2  Choosing a City

2.1 Introduction

The package template described throughout this guide is designed for cities that use the Socrata Open Data Platform. Because Socrata provides a standardized API across portals, the same package architecture can be reused with only a handful of city-specific modifications.

This chapter lists major U.S. cities with Socrata-based Open Data portals and provides the two most important values needed when creating a new package:

  • the Socrata portal domain
  • the metadata catalog dataset used by city_list_datasets()
TipUsing this table

Each row contains the two values needed to begin creating a new package.

  1. Portal Domain
  • Used by .city_endpoint() in utils_request.R.
  • For example: https://data.sfgov.org/resource/
  1. Metadata Dataset
  • Used as the dataset_id in city_list_datasets().
  • For example: y8fp-fbf5

If your city is not listed here, search for its Open Data portal and determine whether it is powered by Socrata. If so, identify:

  • the portal domain
  • the metadata catalog dataset

Those two values are all that is required to begin adapting the package template.
